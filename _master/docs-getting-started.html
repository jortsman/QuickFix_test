Quick Fix Grid

Introduction

Grid systems are used for creating page layouts through a series of rows (grid grid-tab) and columns that house your content. Here's how the Quick Fix grid system works:

  • Rows must be placed within a .grid grid-tab for proper alignment and padding.
  • Use rows to create horizontal groups of columns.
  • Content should be placed within columns
  • Predefined grid classes like .col-12-12 are available for quickly making grid layouts. Less mixins can also be used for more semantic layouts.
  • Grid columns are created by specifying the number of twelve available columns you wish to span. For example, three equal columns would use three .col-4-12
