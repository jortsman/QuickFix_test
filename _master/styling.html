Framework QuickFix uses Khula family of google fonts. The framework uses rems with a base font of 16px. In addition to the basic six headings, it provides a .font-large class for extra large font size.

Other elements

For anchors use the a tag

For small use the small tag

For bold use the strong tag

For italics use the em tag

For highlighting use the mark tag

For abbreviations and acronyms use the abbr tag

for addresses use the address tag
When you want to use a quote from a different source use the blockquote tag.
